NBt-Crafting

Create a Recipe for the Crafting Table which has an Item with NBT Data as Result.

You just need to create a normal Recipe in JSON format (Generator), the Item with NBT which will be the output and define the Minecraft Version. The Count of your output Item will be automatically defined from the Recipe.

But be aware that the result Item can't be an Item which could be in the Inventory with NBT on it, because the defined count of it will be cleared after the player crafts the Item.
I recommend to use something like Barriers, Structure Voids or Knowledge Books (for non stackable Items).

To get the Datapack you just need to fill out all fields which are marked with needed. To add another Recipe you can use the Button in the Navbar or the shortcut Ctrl + Enter. The Sidebar on the left can be used to navigate between each Recipe by clicking on it.

Even more options can be found under the Advanced options Spoilers. The gray placeholder Text will be used in the final Pack if you don't overwrite it.
You can use the $ character in File Name Fields to keep the same Name in new Recipes. This will only work correctly if you change the Folder Path. If you use a $ character in Folder Path Fields it will be replaced with the Recipe Name. The Recipe Name can be defined if you open the first Advanced options Spoiler and check the checkbox in it. Afterwards you can change it for each Recipe individual.

Because a normal Recipe Unlocker Datapack would break this Crafting Methode, you should use this one.
