Gateway

1.16 +

This Datapack adds the Ability to travel very fast between places you've already visited. You can even mark each Gateway with a Block to remember easily which Gateway you need to take.

Setup
Craft two Gateways and a Gateway Linker. Place the Gateways at the two Endpoints and right click bought Gateways with the Gateway Linker. If a Portal isn't linked, it will cry, because it hasn't a partner ; ) .
To mark the Gateway you need to drop a Flint and the marking Block underneath the Gateway. Only the Flint will be consumed.

To remove the Gateway just break the Block underneath it. If the Gateway was linked to another, the linked Gateway will be removed as well.

Crafting

Gateway Linker Recipe

Gateway Recipe

The Crafting Table will show a Eye of Ender or a Crying Obsidian as result, but you will get the correct Item after you take it out.

You can repair your Gateway Linker, if you drop a Ender Pearl with the Gateway Linker on an Anvil. To get more Durability you can also enchant it with Unbreaking.

Settings

Change the maximum Count of linked Gateways you can have in the World. (Default: 20)
It can cause lags, if you have a to big amount of Gateways.

/scoreboard players set #max gateway 20

Disable Tool Damage for the Gateway Linker.
To enable it again you need to execute the same Command except of the 1 replace with a 0.

/scoreboard players set #damage gateway 1

Remove all Gateways.

/function gateway:reset
