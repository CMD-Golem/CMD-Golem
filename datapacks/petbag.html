PetBag

1.14 +

This Datapacks adds a Chest in which you can store your Items and follows you like a dog.

If you don't want, that to PetBag follows you anymore, you can sneak on it. To make it following you again, you need to sneak on it again.

Crafting

Drop these Items on the same Place:

  • 5x Leather
  • 1x Ender Chest
  • 3x Iron Block

Setup

  1. Install the Datapack and rejoin the World or restart the Server.
  2. Go to the Spawn point of the World and execute the following Command:
    /trigger respawn

To remove all PetBags you need to execute the Command above again. Afterwards you need to do Setup again.
