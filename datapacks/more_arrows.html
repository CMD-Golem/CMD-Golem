More Arrows

1.14 +

This Datapacks adds a lot new Arrows. Like TNT Arrow, Teleport Arrow and so on.

It also changes the way to craft Tipped Arrows. Now the crafting for them works like the crafting for the new Arrows. You need to drop the normal Potion or the Splash Potion on the Fletching Table and shoot it with a Bow.

Crafting

Drop the matching Item, which you can see in the Table underneath in the Arrow Spoiler, on a Fletching Table and shoot the Item with a Bow or Crossbow. (You can't use an Infinity Bow and you need to be in survival or adventure Mode)

If you have OP or cheats enabled you can get all Arrows with the following Command:

/function more_arrows:give
